Modules

Add your own class in modules directory, and use it in the code by put use directive.
This will help you to reuse your code, from any Controller or Model.

Example:

Using CSVFileReader
We included sample module CSVFileReader class in Modules\Common\CSVFileHandler directory,
This class have one static method Read, to read CSV file and will output the header and list data in array,
to get the header : $result["header"],
to get the list data : $result["data"],
sample code :

    use Modules\Common\CSVFileHandler\CSVFileReader;

    $result = CSVFileReader::Read($csvfilepath, true);

    $header = $result["header"];
    $listdata = $result["data"];
